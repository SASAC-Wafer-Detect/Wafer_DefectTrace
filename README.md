# Wafer\_DefectTrace

반도체 엔지니어가 별도의 AI 전문 지식 없이, 웨이퍼 이미지를 도구에 넣기만 하면 어떤 불량 패턴인지 자동으로 분류하고 어느 공정에서 문제가 발생했을 가능성이 높은지 즉시 확인할 수 있는 진단 시스템을 제공합니다.



\## 설치 방법



\# 1. torch 먼저 설치 (GPU)

pip install torch==2.4.1+cu121 torchvision==0.19.1+cu121 --index-url https://download.pytorch.org/whl/cu121



\# 2. 나머지 패키지 설치

pip install -r requirements.txt
